# TDD and Python FizzBuzz Kata

>Practicing TDD with Python

## Dependencies

- Python 3.7.0,  Check version, `python -V`
- PyTest, 
  - Install pytest, `pip install pytest`
  - Check pytest version, `pytest --version`
